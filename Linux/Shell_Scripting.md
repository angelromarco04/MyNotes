# Parameters
```bash
$promt> <filename> <param1> <param2> ...
```

- Params. can be accessed with `$i` being `i` the param number (`$1`, `$2).
- `$0` is always the filename.
- `$#` is the number of parameters specified.
- `shift` command removes the first arg
