## Process Typology

- **CPU intensive**
	- More CPU use than I/O.
	- Few CPU strikes with long duration.
```mermaid

    title CPU intensive
    dateFormat X

```
- **I/O intensive**
	- More I/O than CPU use.
	- Many CPU strikes with short duration.
