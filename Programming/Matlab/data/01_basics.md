### Basic operations

| Name | Symbol |
| ---- | ---- |
| Parenthesis | `( )` |
| Power | `^` |
| Addition | `+` |
| Substraction | `-` |
| Multiplication | `*` |
| Division | `/` |

## Variables

```matlab
% Variables
a1 = 1

% Variables simbolicas
syms x y a2 = x + 3*y

% Variables predefinidas
ans Ultimo resultado no asignado a variable
pi Valor pi
i j Unidades imaginarias
inf Infinito
NaN Not a number
date Fecha
```

## Some basic commands

| Command | Description |
| ---- | ---- |
| `whos` | Displays all variables in current session. |
| `clear` | Remove all variables from current session. |
