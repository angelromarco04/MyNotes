## Introduction to tags

- There are 2 types of tags:
	- Open/Close tags.
	```HTML
	<tag>contents</tag>
	```
	- Standalone tags.
	```HTML
	<tag>
	```

- Tags can also have some attributes
```Html
<tag attribute="something">
```
## Page Structure

```html
<!DOCTYPE html>
<html lang="en">
	<head>
		<!-- Not displayed information -->
	</head>
	<body>
		<!-- Content of the webpage -->
	</body>
</html>
```
